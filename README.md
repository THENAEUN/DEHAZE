# DEHAZE

Vision Transformer 아키텍처를 활용해 이미지의 안개를 깨끗하게 제거하고 원래의 색감을 복원하는 딥러닝 프로젝트입니다. 통제된 환경의 단일 이미지 안개 제거를 시작으로, 향후 실제 야외 안개 환경과 연속된 영상 데이터로 인공지능 모델의 파이프라인을 확장해 나가는 것을 목표로 합니다.

## Overview
자율주행, CCTV 모니터링 등 다양한 컴퓨터 비전 시스템에서 '안개'는 시야를 가리고 객체 인식을 방해하는 치명적인 위험 요소입니다.
기존의 안개 제거 모델들은 안개를 지우는 데만 집중한 나머지 원래의 색감이 흑백처럼 변하거나(색상 왜곡), 4K 이상의 고해상도 이미지를 처리할 때 그래픽카드 메모리가 터져버리는(OOM) 한계가 있었습니다.

본 프로젝트는 이러한 문제를 직접 커스텀한 네트워크 구조와 손실 함수를 통해 극복하고, 나아가 사진 한 장을 넘어 연속된 영상 속의 안개까지 실시간으로 걷어내는 통합 AI 모델을 개발하고자 시작되었습니다.

- Input (Hazy): 안개가 낀 고해상도 원본 이미지
- Output (Dehazed): 색상 왜곡 없이 맑게 복원된 이미지
- Metrics: 각 이미지별 객관적 지표 (PSNR, SSIM, Runtime) 자동 추출

## Troubleshooting & Customizing
- 안개가 걷히면서 색감이 이상해지는 문제 (Color Distortion)
    - 문제: 단순한 오차 계산법($L_1$ Loss)만 쓰면 물빠진 색감이 나오는 현상 발생.
    - 해결: 모델이 정답 이미지와 예측 이미지의 RGB 색상 벡터 각도를 비교하여, 원래의 색감을 유지하도록 색상 보존 손실(Color Loss) 메커니즘을 코드에 추가했습니다.

- 고해상도 메모리 초과 문제 (CUDA OOM)
    - 문제: 40GB 이상의 VRAM을 가진 GPU에서도 초고해상도 이미지를 통째로 넣으면 메모리가 초과되어 프로그램이 뻗어버림.
    - 해결: 이미지를 일정 크기(Patch)로 잘라서 순차적으로 안개를 제거한 뒤, 다시 감쪽같이 이어 붙이는 패치 기반 추론(Patch-based Inference) 코드를 작성하여 메모리 문제를 우회했습니다.

- 모델 구조 개선
    - 기존 방식의 한계를 보완하고 복원 품질을 높이기 위해, Vision Transformer 기반의 AdvancedDehazeFormer를 새롭게 도입했습니다.

## Roadmap
이 프로젝트는 다음 3단계로 점진적인 성능 향상과 적용 범위 확장을 진행 중입니다.

- 단일 이미지 안개 제거 (Single Image Dehazing)
- 실제 안개 사진 적용 (Real-world Haze)
- 영상 안개 제거 (Video Dehazing)

## Process & Evaluation
- 테스트 환경: O-HAZE 데이터셋 43장 테스트셋 고정 적용
- 매칭 방식: 정규표현식을 이용한 숫자 식별자 기반 Hazy-GT 자동 매칭(Robust Matching)
- 결과 자동화: 성능 비교를 위해 모델 실행 시 PSNR, SSIM, Runtime(초) 지표를 자동으로 추출하여 summary_results.csv와 per_image_results.csv 두 가지 형태의 리포트로 자동 저장되도록 파이프라인을 구성
